HTML

Hypertext Markup Language

เป็นภาษาคอมพิวเตอร์ประเภทมาร์กอัปที่ใช้ในการสร้างเว็บเพจ โดยมาร์กอัปคือการแทรกข้อความหรือคำสั่งพิเศษเข้าไปในข้อความปกติเพื่อบอกให้เว็บเบราว์เซอร์แสดงผลข้อความนั้นออกมาในรูปแบบที่ต้องการ

HTML ประกอบด้วยแท็ก (tag) ซึ่งเป็นเครื่องหมายกำกับที่ใช้ในการกำหนดรูปแบบการแสดงผลของข้อความและองค์ประกอบต่างๆ บนเว็บเพจ เช่น หัวเรื่อง ย่อหน้า ลิงก์ รูปภาพ ตาราง ฯลฯ แท็ก HTML ประกอบด้วยชื่อแท็กและปิดแท็ก โดยชื่อแท็กจะบอกให้เว็บเบราว์เซอร์ทราบว่าแท็กนี้ทำหน้าที่อะไร ปิดแท็กจะบอกให้เว็บเบราว์เซอร์สิ้นสุดการแสดงผลคำสั่งในแท็กนั้น

HTML Terms

  • Internet เครือข่ายคอมพิวเตอร์ที่ใหญ่ที่สุดในโลก เกิดจากการเชื่อมโยงของเครือข่ายต่างๆ เข้าด้วยกัน
  • Hypertext รูปแบบเอกสารที่บรรจุการเชื่อมโยงไปยังเอกสารอื่นๆ ซึ่งสามารถใช้ข้อความ หรือรูป เป็นจุดเชื่อมโยง
  • WWW ย่อจาก World Wide Web เป็นการสื่อสารด้วยการเชื่อมโยงเครือข่ายข่าวสารแบบใยแมงมุม(Web) แสดงผลด้วยเอกสารไฮเปอร์เท็กซ์
  • HTTP ย่อมาจาก Hypertext Transfer Protocol เป็นรูปแบบการสื่อสารที่ใช้ในการรับส่งข้อมูลไฮเปอร์เท็กซ์ในเครือข่ายอินเทอร์เน็ต
  • Web Browser โปรแกรมสำหรับแสดงผลหน้าเว็บ เช่น Internet Explorer, Mozilla Firefox และ Google Chrome เป็นต้น
  • Web Page หน้าเอกสารที่อยู่ในรูปของไฮเปอร์เท็กซ์
  • Web Site กลุ่มของหน้าเว็บหลายๆ หน้ารวมเข้าด้วยกัน
  • Home Page หน้าเว็บ หน้าแรกของเว็บไซต์
  • Web Site เครื่องให้บริการที่เป็นที่เก็บข้อมูลของ เว็บไซต์

HTML เป็นภาษาคอมพิวเตอร์ที่มีบทบาทสำคัญในการสร้างเว็บไซต์ในยุคปัจจุบัน โครงสร้างของ HTML ประกอบด้วย Tag ต่างๆ ที่ใช้ในการกำหนดโครงสร้างและเนื้อหาของหน้าเว็บไซต์ โดย Web Browser จะแปลง Tag เหล่านี้เป็นหน้าตาของเว็บไซต์ที่เราเห็นบนหน้าจอ เช่น การกำหนดหัวเรื่องด้วย <h1> หรือการแทรกรูปภาพด้วย <img> ฯลฯ

HTML มีความสามารถในการเชื่อมโยงข้อมูลระหว่างหน้าเว็บไซต์ต่างๆ ทำให้การเชื่อมโยงข้อมูลในอินเทอร์เน็ตเป็นไปอย่างสะดวกและรวดเร็ว นอกจากนี้ HTML ยังมีคุณสมบัติในการแก้ไขเนื้อหาของหน้าเว็บไซต์ที่มีอยู่แล้วได้โดยไม่ต้องใช้โปรแกรมเสริม

การเรียนรู้ HTML มีความสำคัญเนื่องจากมันเป็นเครื่องมือสำคัญในการสร้างและแก้ไขเว็บไซต์ แม้ว่าโปรแกรมประยุกต์สำหรับสร้างหน้าเว็บอาจมีมากมาย แต่การเรียนรู้ HTML ยังช่วยให้ผู้ใช้สามารถเข้าใจโครงสร้างพื้นฐานและปรับแต่งหน้าเว็บไซต์ตามต้องการได้โดยไม่ต้องพึ่งพาโปรแกรมเสริมหรือผู้อื่นในการทำเช่นนี้

โครงสร้างพื้นฐานของ HTML

โครงสร้างของ HTML จะประกอบไปด้วยส่วนของคำสั่ง 2 ส่วน คือ ส่วนที่เป็น ส่วนหัว (Head) และส่วนที่เป็นเนื้อหา (Body) โดยมีรูปแบบคำสั่งดังนี้

<HTML>
<HEAD>
<TITLE> ชื่อโปรแกรมหรือข้อมูลที่ต้องการแสดงในส่วนหัว </TITLE>
</HEAD>
<BODY>
คำสั่งหรือข้อความที่ต้องการให้แสดง
</BODY>
</HTML>
